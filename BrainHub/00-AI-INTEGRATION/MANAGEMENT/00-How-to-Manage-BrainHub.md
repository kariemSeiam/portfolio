### **New Project**

**Steps:**
1. **Deep Analysis First:**
   - Analyze project repository/files thoroughly
   - Understand architecture, tech stack, features
   - Identify patterns, relationships, significance

2. **Create/Update Project Deep Dive:**
   - Create file in `03-Projects-Deep-Dive/[Category]/` (if new major project)
   - Or update existing entry in category file (if part of series)
   - Include: Tech stack, architecture, features, why notable

3. **Update AI-INTEGRATION Context Files (Critical for AI):**
   - Update `00-AI-INTEGRATION/02-Projects-Context.md` (detailed project entry)
   - Update `00-AI-INTEGRATION/01-Core-Context.md` (if flagship/major project)
   - Update `00-AI-INTEGRATION/03-Tech-Stack-Context.md` (if new tech/patterns)

## ğŸ” Analyzing Existing Projects

### **Deep Project Analysis**

**When:** You want to add accurate details about an existing project (especially if details are incomplete or outdated)

**Tools & Methods:**

1. **Repository Analysis (GitHub/Public Repos):**
   - Use GitHub MCP tools or direct repository access
   - Explore repository structure and files
   - Read key files: README.md, package.json, build.gradle, requirements.txt, etc.
   - Examine source code structure (packages, modules, architecture)
   - Review commit history for timeline insights
   - Check dependencies and tech stack from config files

2. **Codebase Analysis:**
   - Examine directory structure
   - Review key source files (MainActivity, app.py, etc.)
   - Identify architecture patterns (MVVM, MVC, etc.)
   - Understand tech stack from dependencies
   - Note package names, versions, frameworks

3. **Documentation Review:**
   - README files
   - Documentation directories
   - Configuration files
   - Comments and code structure

**Output:**
- Accurate tech stack (with versions)
- Architecture patterns identified
- Package/application names
- Key features and structure
- Timeline information (from commits)
- Relationships to other projects

**Then:** Follow "New Project" or "Project Updates" steps with accurate information
