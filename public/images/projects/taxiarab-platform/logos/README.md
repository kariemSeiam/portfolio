# Taxiarab Platform App Logos

Place app logo images here for the Taxiarab Platform apps.

## Required Files

1. **rider-logo.png** - Logo for the Rider App (تاكسي عرب)
   - Recommended size: 512x512px or 1024x1024px
   - Format: PNG with transparency
   - Square aspect ratio

2. **driver-logo.png** - Logo for the Driver App (عرب للسائقين)
   - Recommended size: 512x512px or 1024x1024px
   - Format: PNG with transparency
   - Square aspect ratio

## How to Add Logos

1. Download the app icons from Google Play Console or extract from APK
2. Resize to 512x512px or 1024x1024px
3. Ensure transparent background
4. Save as PNG format
5. Place files in this directory:
   - `rider-logo.png`
   - `driver-logo.png`

The logos will automatically display in the project modal's app download cards.
